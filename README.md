# Raveen Kumar Portfolio

Static HTML portfolio website prepared for GitHub Pages.

## Files

- `index.html` is the live homepage GitHub Pages will serve.
- `styles.css` contains the shared styling for the homepage and project pages.
- `projects/` contains one project detail page per selected CV project.
- `index2.html` is the original draft kept as a backup.
- Image files in this folder are used by the page and should stay in the repository root.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder, including `index.html` and the images.
3. Open the repository settings.
4. Go to **Pages**.
5. Set the source to **Deploy from a branch**.
6. Choose the `main` branch and `/ (root)`.
7. Save, then open the Pages URL GitHub provides.

If the site does not update immediately, wait a minute and refresh the GitHub Pages URL.
