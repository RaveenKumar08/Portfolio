Clutch Modelling in MATLAB

Mathematical simulation of clutch engagement, slip transition, and gear-ratio influence.

Back to Projects

Overview

This MATLAB project models clutch engagement using a mathematical driveline model. The simulation studies the transition from slip to the no-slip condition, where the input and output angular velocities become equal after engagement. The gear-ratio values are based on assumptions and may not represent real-world applications. The project was done to understand clutch behaviour through a mathematical model.

Work Performed

  • Defined a clutch engagement model around the no-slip condition, where omega 1 equals omega 2.
  • Simulated clutch response across five transmission gear ratios.
  • Compared slip time and vehicle velocity response for each gear case.
  • Prepared result plots and a table for model interpretation.
