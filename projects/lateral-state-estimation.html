Vehicle Lateral Dynamics

Lateral Dynamics - State Estimation

Vehicle sideslip estimation using model-based, kinematic, washout-filtered, and SCKF approaches.

Back to Projects

Overview

This project focused on estimating vehicle body sideslip, a key lateral dynamics state that is difficult to measure directly in normal vehicle setups. The work compared model-based, kinematic, washout-filtered, and SCKF approaches using measured vehicle signals while keeping detailed datasets and parameter values out of the public project page.

The study used multiple driving manoeuvres to evaluate how each estimator behaves under steady-state cornering, transient steering, low-speed motion, and higher-speed dynamics. This made the project less about one perfect estimator and more about understanding where each method becomes reliable or limited.

Work Performed

  • Built a bicycle-model estimator to calculate lateral velocity and body sideslip from vehicle and steering inputs.
  • Implemented a kinematic estimator using lateral acceleration, yaw rate, longitudinal velocity, and sensor-position corrections.
  • Combined model-based and kinematic estimation through a washout filter with a tunable time constant.
  • Compared estimator behaviour across several manoeuvre types using error trends rather than exposing full measurement data.
  • Implemented an SCKF workflow with vehicle states, measured outputs, steering input, process noise, and measurement noise.
  • Reviewed estimator sensitivity to manoeuvre type, vehicle speed, sensor drift, and tyre nonlinearity.

Key Insights

  • Model-based estimation is strongest in steady-state conditions where the bicycle model assumptions remain reasonable.
  • Kinematic estimation can follow transient manoeuvres more naturally, but integration makes it vulnerable to drift from small sensor bias.
  • Washout filtering is useful because it blends the strengths of both approaches, though the filter tuning is manoeuvre dependent.
  • Estimator accuracy changes with vehicle speed because higher speeds can amplify the effect of steering inputs and tyre nonlinearities.
  • A single estimator setting is rarely optimal for every manoeuvre, so validation across multiple driving cases is essential.

Summary

The project shows that sideslip estimation is a trade-off between model confidence and measurement behaviour. Model-based methods are clean and interpretable, kinematic methods capture fast changes but drift, and washout filtering gives a practical bridge between them. The SCKF implementation extends the work toward a more complete state estimation framework for lateral vehicle dynamics.
