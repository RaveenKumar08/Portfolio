Overview

This project involved contribution to a modular aircraft prototype for defence applications. As part of the structures team, the work focused on fuselage and wing component design using Onshape CAD.

Work Performed

  • Supported fuselage component design as part of the structures team.
  • Contributed to wing component design and modular aircraft layout work.
  • Used Onshape for CAD modelling and collaborative design updates.
  • Worked within a project context involving collaboration with FMV.

Challenges

Designing within a collaborative aircraft layout

Challenge: Fuselage and wing components needed to fit a modular prototype architecture while staying aligned with other structure-team updates.

Solution: Used Onshape collaboration workflows to keep component geometry connected to the shared aircraft layout and update design details as interfaces changed.

Showing the work without exposing restricted details

Challenge: The public portfolio page needs to describe the engineering contribution without sharing confidential or unapproved project material.

Solution: Kept the page focused on role, methods, and design responsibility, with CAD visuals reserved until approved for public use.
