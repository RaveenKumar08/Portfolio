Vehicle Longitudinal Dynamics

Longitudinal Dynamics - Slip Control

ABS and TCS controller development for managing wheel slip during braking and traction events.

Back to Projects

Overview

This project studied longitudinal tyre slip control for both traction and braking scenarios. The work connected tyre force limits, driver input, wheel slip behaviour, and controller response to show how ABS and TCS can keep the vehicle close to useful grip regions without publishing full calibration values or detailed model data.

The simulation work used a vehicle dynamics environment with controller logic built around wheel slip feedback. The core objective was to manage slip during acceleration and braking while maintaining a practical balance between longitudinal force generation, stability, and steerability.

Work Performed

  • Reviewed the longitudinal force and lateral force trade-off using the tyre friction circle concept.
  • Analysed tyre slip curves during traction and braking to identify sensible controller activation regions.
  • Built separate ABS and TCS activation logic using wheel slip feedback and driver pedal inputs.
  • Designed controller modes for P, PI, PD, and PID behaviour to compare slip regulation quality.
  • Implemented independent slip control logic for driven and braked wheels in the simulation model.
  • Investigated saturation behaviour and improved the PID loop using anti-windup and derivative filtering.

Challenges

Handling nonlinear saturation

Challenge: Controller output limits can cause integral windup and delayed correction when the actuator saturates.

Solution: Added anti-windup and derivative filtering to make the PID response more realistic and stable in simulation.

Key Insights

  • Slip control is not only about acceleration or braking performance; it also helps preserve steerability by avoiding excessive use of the tyre's available longitudinal force.
  • P control reacts quickly but can leave steady-state error, while adding integral action improves tracking at the cost of greater tuning sensitivity.
  • Derivative action can reduce overshoot, but the final behaviour depends on the interaction between all controller terms and the actuator limits.
  • Anti-windup is important for realistic control loops because saturation can otherwise cause accumulated integral action and unwanted slip correction.
  • Classical tuning is useful for a starting point, but nonlinear, time-varying ABS and TCS cases need simulation-led refinement.

Summary

The project demonstrates a complete control workflow for longitudinal slip: identify the control objective from tyre behaviour, define activation logic, compare controller modes, and refine the loop for saturation. The final insight is that robust slip control depends on both the chosen reference behaviour and the practical details of the controller implementation.
