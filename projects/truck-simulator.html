Truck Simulator

Ergonomic redesign, CAD modelling, and physical validation of a truck driving simulator testbed.

Back to Projects

Overview

This project redesigned an existing compact race-car simulator concept into a truck-oriented simulator testbed. The work focused on truck-specific ergonomics, including the seat and pedal relationship, steering height and inclination, steering reach, and display height for an upright driving posture.

A standalone wooden platform was manufactured first to validate the layout before applying permanent changes to the dynamic simulator.

Work Performed

  • Used a stage-gate workflow to define goals, benchmark the existing simulator, finalise the design, manufacture the testbed, and validate the result.
  • Built the CAD layout in Siemens Solid Edge using Scania cabin geometry as the main ergonomic reference.
  • Selected make-or-buy boundaries for platform, seat, pedals, steering wheel, steering mount, display, and display mount subsystems.
  • Designed and validated the pedal assembly layout using FANATEC hardware and CAD reference geometry.
  • Developed the steering subsystem with a designed mount and slider rail to achieve truck-like height, reach, and inclination.
  • Supported physical manufacturing through wooden platform construction and 3D-printed PETG carbon-fibre-reinforced validation parts.

Challenges

Converting a compact simulator into a truck layout

Challenge: The existing race-car style setup did not match upright truck seating, steering reach, pedal position, or display height.

Solution: Rebuilt the ergonomic package around Scania cabin references and verified the layout with a standalone wooden testbed before final integration.

Packaging real hardware within build limits

Challenge: The design had to work with purchased steering, pedal, display, and seat hardware while remaining manufacturable in the workshop.

Solution: Defined make-or-buy boundaries, created a Solid Edge assembly, and used 3D-printed validation parts to check steering mount geometry.

Engineering Targets

Parameter Achieved Relative to Target
Steering height from floor 103.6%
Steering horizontal position 75.0% to 105.3% adjustment envelope
Steering inclination 100%
Display centerline 100% target-representative placement

Validation and Way Forward

The completed wooden model validated the main spatial layout and demonstrated that the major truck simulator subsystems could be packaged together. The static testbed integrated the Scania truck seat, FANATEC steering and pedal hardware, a raised display, and a truck-like steering mount arrangement.

The study also identified limits in converting a tightly packaged race-car simulator into a truck simulator. Recommended next steps include extending the monitor support, redesigning the steering holder with outer vertical supports to avoid interference, adding better seat-height adjustment, and eventually designing a custom dynamic platform around truck ergonomics from the start.

Tools Siemens Solid Edge, CAD assembly, 3D printing, workshop manufacturing
