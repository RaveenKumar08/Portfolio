Vehicle Vertical Dynamics

Vertical Dynamics - Vibration Control

Ride response analysis using passive suspension, Skyhook control, and H-infinity control strategies.

Back to Projects

Overview

This project studied vertical vehicle vibration control using passive suspension models and active control concepts. The work started from simple single-degree-of-freedom dynamics and progressed toward two-degree-of-freedom and half-car models, comparing how different controllers influence ride response and disturbance rejection.

The project page keeps the public description focused on method, behaviour, and engineering insight. Detailed equations, tuning constants, and full response data remain inside the report rather than being exposed directly on the portfolio page.

Work Performed

  • Derived and simulated single-degree-of-freedom passive suspension dynamics.
  • Extended the study to two-degree-of-freedom dynamics to separate sprung and unsprung mass behaviour.
  • Compared passive damping against PD, PID, Skyhook, and H-infinity control approaches.
  • Evaluated response behaviour under impulse and sinusoidal road disturbance inputs.
  • Built a simple half-car vertical model to include bounce and pitch behaviour.
  • Reviewed controller robustness by varying key suspension parameters from their nominal values.

Key Insights

  • Single-degree-of-freedom modelling is useful for fundamentals, but two-degree-of-freedom modelling gives a clearer view of ride comfort and tyre-road contact behaviour.
  • Skyhook control improved damping and remained comparatively robust when the model parameters were varied.
  • H-infinity control can perform strongly for targeted response requirements, but it is more sensitive to model mismatch.
  • Impulse and sinusoidal road inputs reveal different strengths, so controller selection depends on the disturbance type being prioritised.
  • Controller gains need a practical balance: too little damping is ineffective, while excessive gain can create saturation or control-effort issues.

Summary

The project builds a clear path from passive suspension fundamentals to active vibration control. Skyhook control stood out as a practical and robust damping strategy, while H-infinity control showed the strengths and trade-offs of a more model-dependent design. The main learning was how model fidelity, disturbance type, and controller sensitivity shape vertical dynamics performance.
